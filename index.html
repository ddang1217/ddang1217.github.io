Q1 Create a dataframe containing driver names of instances where free_wine = 1, discount_customer = 1, and the order contained more than 4 pizzas. (There will be repeated names).

  • Your solution should be a dataframe assigned to Q1. In the Console Window the output should look something like this:
  driver
1 [name]
2 [name]
...
n [name]
### Do not edit the following line. It is used by CodeGrade.
# CG Q1 # 

### TYPE YOUR CODE BELOW ###
Q1 <- pizza |>
        filter(free_wine == 1 & discount_customer == 1) |>
        group_by(driver) |>
        filter(pizzas > 4) |>
        select(driver) |>
        as.data.frame()

Q5 What was the highest amount of free wine given by day/driver combination? (For instance, Friday Bruno was 13, while Wednesday Salvator was 12.)
